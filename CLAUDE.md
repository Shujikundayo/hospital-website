## ページ構成（全6ページ）

既存サイトと同じ構成を維持する。

1. `index.html` — トップページ
2. `about.html` — クリニックについて（院長挨拶・理念など）
3. `department.html` — 診療科目
4. `access.html` — アクセス・診療時間
5. `news.html` — お知らせ
6. `contact.html` — お問い合わせ

## デザイン方針

- **基調色:** 水色・緑色・白色を中心とした温かみのある配色
- **印象:** 清潔感・安心感・親しみやすさ
- **フォント:** 日本語フォント優先（Noto Sans JP など）
- **スタイル:** 角丸・柔らかい影・余白を活かしたレイアウト

### カラーパレット（目安）

| 役割 | 色 |
|------|----|
| メインカラー | 水色系 `#5BB8D4` 付近 |
| サブカラー | 緑系 `#6DBF87` 付近 |
| 背景 | 白 `#FFFFFF` / 薄グレー `#F5F9FC` |
| テキスト | 濃いグレー `#333333` |
| アクセント | 暖かみのあるオレンジ・黄緑など |

## レスポンシブ対応（必須）

- **ブレークポイント:** 768px（スマホ/タブレット境界）を基本とする
- モバイルファーストで実装
- ハンバーガーメニュー必須（スマホ時）
- タップしやすいボタン・リンクサイズ（最低 44px）

## コーディング規約

- インデント: スペース2文字
- 文字コード: UTF-8
- HTMLはセマンティックタグを活用（`<header>`, `<main>`, `<section>`, `<footer>` など）
- CSSカスタムプロパティ（変数）でカラーを管理
- JavaScriptはバニラJS（`querySelector` / `addEventListener` ベース）
- 外部依存は最小限（Google Fonts, Font Awesome 程度は可）

## 共通コンポーネント

### ヘッダー

- クリニックロゴ（左）＋ グローバルナビ（右）
- スマホ時はハンバーガーメニュー
- 電話番号を目立つ位置に表示

### フッター

- 所在地・電話番号・診療時間の簡易表示
- コピーライト表記
- ページトップへ戻るボタン

## 注意事項

- 個人情報（患者情報など）を扱うフォームはhttps前提で設計
- お問い合わせフォームはサーバーサイド処理なし（静的サイト）のためFormspree等の外部サービス利用を想定
- アクセスマップはGoogle Maps埋め込みを使用
- 画像は `alt` 属性を必ず記述（アクセシビリティ対応）
